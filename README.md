# Aurelius Metals (Astro + Cloudflare Workers)

Marketing site for **Aurelius Metals** — mobile precious metals buying and appraisals in San Diego.
Built with **Astro** and deployed to **Cloudflare Workers** using Wrangler.

## Tech
- Astro
- @astrojs/cloudflare adapter (Workers)
- Wrangler

## Local development

### Prereqs
- Node.js (LTS recommended)
- Git

### Install & run
```bash
npm install
npm run dev
